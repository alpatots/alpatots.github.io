*With Craig Boucher, Ulf Knoblich, Daniel Miller, Amin Saied, and Venky Venkateshaiah.*

Describes Microsoft's platform for running software experiments at scale
across a highly heterogeneous ecosystem of products (websites, mobile apps,
cloud services, OS) and compute fabrics. Presents a solution for
democratizing metric definition and analysis so that engineers, product
managers, and data scientists can all use the same experimentation
infrastructure regardless of product or platform.
